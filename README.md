# java-algo-test-intro
이 프로젝트는 인프런 강의인 '자바(Java) 알고리즘 문제풀이 입문: 코딩테스트 대비'를 학습하며 알고리즘 문제를 풀고 풀이 과정을 기록하는 프로젝트 입니다.

## 목표
- 매일 최소 하나의 알고리즘 문제 풀기
- Java에 대한 이해도 높이기
- 자료구조/알고리즘에 대한 이해도 높이기

## 사용언어
- **JAVA**
